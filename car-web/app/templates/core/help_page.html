Session

General

A session at the end is a property container which will be send to the server if the user want a word document should be created.

Session Content

  • the figures directory name
  • the name of the default table
  • the name of the template document
  • the selected figures
  • the selected multi figure groups
  • the scaling for single figures and grouped figures
  • the comment text (if predefined by default table for a figure or multi figure)
  • text replacement
